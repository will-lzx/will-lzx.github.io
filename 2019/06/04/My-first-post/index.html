markdown 使用教程

Posted on 2019-06-04 | In tools

This topic is a simple markdown user manual.

标题

在想要设置为标题的文字前添加#, 一级标题即添加一个#, 二级标题添加两个#，以此类推，最多支持六级标题

注：一般在# 后添加一个空格

#这是一级标题

##这是二级标题

字体

  • 加粗
    要加粗的文字左右分别用两*号包起来
  • 斜体
    要斜体的文字左右分别用一个*号包起来
  • 斜体加粗
    要斜体加粗的文字左右分别用三个*号包起来
  • 删除线
    要添加删除线的文字左右分别用～～包起来

引用

在引用的文字前加>即可，也可以嵌套，但是并没有任何意义
